Project 2

Execution instructions

ftserver:
To execute ftserver,
1) Ensure that ftserver and makefile are located in the current working directory
2) Type make to compile
3) Execute ./ftserver [PORTNUM]

ftclient.py:
To execute ftclient.py,
1) Ensure that ftclient.py is located in the current working directory
2) Ensure that ftclient.py is set as executable (e.g., chmod u+x ./ftclient.py)
3) Execute ./ftclient.py [SERVER_HOST] [SERVER_PORT] [-g FILENAME | -l] [DATA_PORT]
